# 量化交易领域

## 领域综述

本领域涵盖量化交易系统的核心功能，包括期货品种筛选、K 线形态识别、交易信号生成、风险控制和回测评估。系统基于 Price Action 理论，不依赖技术指标，专注于价格结构和形态分析。

## 核心流程

### 单品种回测流程
1. 数据获取 → 策略分析 → 信号生成 → 风控评估 → 回测记录
2. 支持新浪期货接口和 CSV 文件两种数据源

### 多品种回测流程
1. 波动性筛选（ATR + 成交量）→ 多品种数据获取 → 并行回测 → 综合评分排名
2. 支持日内/波段两种交易模式

## 技术方案总结

| 组件 | 技术选型 | 说明 |
|------|----------|------|
| 数据获取 | AkShare + 合约代码映射 | 使用 12 月合约获取完整年度数据 |
| 策略分析 | Price Action 理论 | 趋势判断 + 形态识别 + 关键位 |
| 风控 | ATR 止损止盈 | 2×ATR 止损，6×ATR 止盈（3:1 盈亏比） |
| 回测 | 顺序模拟 | 事件驱动交易评估 |
| 评分 | 固定基准归一化 | 收益率40% + 胜率20% + 风险调整25% + 交易次数15% |

## Feature 附录

### feature_20260402_F001_multi-symbol-intraday

**摘要:** 多品种日内交易，支持33个期货品种的波动性筛选和批量回测

**关键决策:**
- 数据获取方式: 使用 AkShare 期货合约代码（如 rb2512）映射获取完整年度数据
- 波动性指标: ATR × 合约乘数（日均波动金额）+ ATR/收盘价（波动率）双重排序
- 交易模式分离: `trading_mode="intraday"`（日内平仓）vs `"swing"`（持仓过夜）
- 综合评分算法: 固定基准归一化（收益率40% + 胜率20% + 风险调整25% + 交易次数15%）
- 品种筛选参数: 波动率区间 [0.8%~5%] + 成交量阈值 10000
- 数据缓存: FuturesDataCache 类支持内存缓存避免重复请求
